 ix.data.Set(key, value[, bGlobal=false[, bIgnoreMap=false]])
View source »
Populates a file in the data/helix folder with some serialized data.

Parameters:

  • key string Name of the file to save
  • value Some sort of data to save
  • bGlobal boolean Whether or not to write directly to the data/helix folder, or the data/helix/schema folder, where schema is the name of the current schema. (default false)
  • bIgnoreMap boolean Whether or not to ignore the map and save in the schema folder, rather than data/helix/schema/map, where map is the name of the current map. (default false) 
 ix.data.Get(key, default[, bGlobal=false[, bIgnoreMap=false[, bRefresh=false]]])
View source »
Retrieves the contents of a saved file in the data/helix folder.

Parameters:

  • key string Name of the file to load
  • default Value to return if the file could not be loaded successfully
  • bGlobal boolean Whether or not the data is in the data/helix folder, or the data/helix/schema folder, where schema is the name of the current schema. (default false)
  • bIgnoreMap boolean Whether or not to ignore the map and load from the schema folder, rather than data/helix/schema/map, where map is the name of the current map. (default false)
  • bRefresh boolean Whether or not to skip the cache and forcefully load from disk. (default false)

Returns:

    Value associated with the key, or the default that was given if it doesn't exists
