 ix.util.StripRealmPrefix(name)

Removes the realm prefix from a file name. The returned string will be unchanged if there is no prefix found.

Parameters:

 ix.util.FindPlayer(identifier[, bAllowPatterns=false])
View source »
Attempts to find a player by matching their name or Steam ID.

Parameters:

  • identifier string Search query
  • bAllowPatterns boolean Whether or not to accept Lua patterns in identifier (default false)

Returns:

    player Player that matches the given search query - this will be nil if a player could not be found
